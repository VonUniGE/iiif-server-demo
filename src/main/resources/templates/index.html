IIIF Server Demo

This is a standalone demo server demonstrating the usage and features of IIIF Image API Java Library and IIIF Presentation API Java Library (Java implementations of the IIIF specifications) and showcasing different IIIF viewers. You can have a look at its code as reference implementation for using the IIIF API Java Libraries.
